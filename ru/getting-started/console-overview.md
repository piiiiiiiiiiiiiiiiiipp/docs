# Ориентация в консоли Cloud Native Solutions

Этот раздел поможет разобраться с интерфейсом консоли: где что находится и как перемещаться между разделами.

---

## Навигация в консоли

В консоли используется **боковая панель** (сайдбар) слева для навигации между разделами.

![img_5.png](img_5.png)

Сайдбар разделён на две группы:

**ПРОЕКТЫ**

| Раздел | Описание |
|---|---|
| **Проекты** | Список проектов организации, создание новых проектов |

**УПРАВЛЕНИЕ** 

| Раздел | Описание |
|---|---|
| **AWS** | Интеграция с Amazon Web Services (Превью) |
| **CDN** | Сервис доставки контента |
| [**Tencent**](./tencent-onboarding.md) | Интеграция с Tencent Cloud |
| **Sangfor** | Сервис Sangfor HCI |

В нижней части сайдбара:

| Раздел | Описание |
|---|---|
| **Биллинг** | Платёжный аккаунт, счета, история операций |
| **Поддержка** | Тикеты и планы поддержки |
| **Настройки** | Настройки организации, управление доступом, безопасность |
| **API** | Токены персонального доступа и OAuth-приложения |

В верхней части сайдбара также находится кнопка **Создать** — используйте её для быстрого создания ресурсов.

**Переключатель организации**, **переключатель языка** (RU / EN) и **аватар пользователя** расположены в правом верхнем углу экрана.

## Обзор основных разделов

### Проекты

Управление облачными проектами. **Проект** — это логическая группировка ресурсов внутри Организации. Каждый проект имеет собственные ресурсы, правила доступа и биллинговую атрибуцию.

![img_7.png](img_7.png)

### AWS

Интеграция с **Amazon Web Services**. В данный момент находится в режиме **Превью** — функциональность доступна для раннего тестирования, но ещё не выпущена в общий доступ.

<!-- скриншот: страница AWS -->

### CDN

**Content Delivery Network** (сеть доставки контента) — распределяет ваш контент по граничным узлам для более быстрой доставки конечным пользователям по всему миру.

<!-- скриншот: страница CDN -->

### Tencent

Интеграция с **Tencent Cloud**. На странице обзора отображается статус подключения вашей Организации к сервисам Tencent Cloud.
[Как зарегистрироваться в **Tencent Cloud**](./tencent-onboarding.md)
<!-- скриншот: страница Tencent -->

### Sangfor

**Sangfor HCI** (Hyper-Converged Infrastructure) — решение для виртуализации и хранения данных для локальных и гибридных развёртываний.

<!-- скриншот: страница Sangfor -->

### Биллинг

Раздел Биллинга — здесь управляется всё финансовое взаимодействие с платформой.

![img_8.png](img_8.png)

Раздел включает следующие вкладки:

| Вкладка | Описание |
|---|---|
| **Информация об аккаунте** | Обзор платёжного аккаунта: название, ID, тип (физическое или юридическое лицо), статус и способ оплаты |
| **Платёжные аккаунты** | Управление платёжными аккаунтами |
| **Счета** | Просмотр и загрузка счетов |
| **История операций** | Полный журнал всех списаний и платежей |
| **Документы** | Юридические и финансовые документы |

Карточка на дашборде показывает **предполагаемый баланс** за текущий месяц в разбивке:
- **Применённые кредиты** — промо- или реферальные кредиты
- **Предоплата** — средства, загруженные авансом
- **Общее использование** — сумма потребления за месяц

### Поддержка

Раздел Поддержки предоставляет доступ к:

- **Тикетам** — создание и отслеживание обращений в поддержку
- **Планам поддержки** — доступны три уровня:
  - **Базовый** — доступ к документации и сообществу
  - **Стандарт** — стандартный SLA по времени ответа для бизнес-использования
  - **Корпоративный** — выделенная поддержка для крупных клиентов
- Ссылка на страницу статуса платформы

![img_9.png](img_9.png)

### Настройки

Управление конфигурацией организации.

![img_10.png](img_10.png)

Раздел Настроек содержит несколько вкладок:

| Вкладка | Описание |
|---|---|
| **Организация** | Название организации, ID и управление участниками |
| **Управление доступом** | Ролевое управление доступом для участников организации |
| **Безопасность** | Настройки безопасности |
| **Рефералы** | Реферальная программа |
| **Предпросмотр функций** | Ранний доступ к новым возможностям |

Во вкладке **Организация** отображается список всех участников с их ролями. Отсюда можно пригласить новых участников в команду.

### API

Управление программным доступом к платформе.

![img_11.png](img_11.png)

| Вкладка | Описание |
|---|---|
| **Токены** | Токены персонального доступа — используются для API-аутентификации |
| **OAuth-приложения** | Регистрация OAuth-приложений |
| **Авторизованные приложения** | Просмотр и отзыв доступа для сторонних приложений |
