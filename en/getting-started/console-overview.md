# Console Overview — Cloud Native Solutions

This section helps you navigate the console interface: what's where and how to move between sections.

---

## Console Navigation

The console uses a **sidebar** on the left side of the screen for navigation.

![img_2.png](img_2.png)

The sidebar is divided into two groups:

**PROJECTS**

| Section | Description |
|---|---|
| **Projects** | List of organization projects, create new projects |

**MANAGEMENT**

| Section | Description |
|---|---|
| **AWS** | Amazon Web Services integration (Preview) |
| **CDN** | Content Delivery Network service |
| [**Tencent**](./tencent-onboarding-en.md) | Tencent Cloud integration |
| **Sangfor** | Sangfor HCI service |

At the bottom of the sidebar:

| Section | Description |
|---|---|
| **Billing** | Billing account, invoices, payment history |
| **Support** | Support tickets and plans |
| **Settings** | Organization settings, access control, security |
| **API** | Personal access tokens and OAuth applications |

At the top of the sidebar you will also find the **Create** button — use it to quickly create new resources.

The **organization switcher**, **language switcher** (RU / EN), and **user avatar** are located in the top-right corner of the screen.

## Main Sections Overview

### Projects

Manage your cloud projects. A **Project** is a logical grouping of resources within your Organization. Each project has its own resources, access rules, and billing attribution.

![img_13.png](img_13.png)

### AWS

Integration with **Amazon Web Services**. Currently in **Preview** mode — available for early testing but not yet generally available.

<!-- screenshot: AWS page -->

### CDN

**Content Delivery Network** — distributes your content across edge nodes for faster delivery to end users globally.

<!-- screenshot: CDN page -->

### Tencent

Integration with **Tencent Cloud**. The overview page shows the connection status between your Organization and Tencent Cloud services.
[How to register to **Tencent Cloud**](./tencent-onboarding-en.md)
<!-- screenshot: Tencent page -->

### Sangfor

**Sangfor HCI** (Hyper-Converged Infrastructure) — a virtualization and storage solution for on-premises and hybrid deployments.

<!-- screenshot: Sangfor page -->

### Billing

The Billing section is where you manage all financial aspects of your account.

![img_14.png](img_14.png)

The section includes the following tabs:

| Tab | Description |
|---|---|
| **Account Information** | Billing account overview: name, ID, type (individual or legal entity), status, and payment method |
| **Payment Accounts** | Manage your payment accounts |
| **Invoices** | View and download invoices |
| **Transaction History** | Full log of all charges and payments |
| **Documents** | Legal and financial documents |

The dashboard card shows your **estimated balance** for the current month, broken down into:
- **Applied Credits** — promotional or referral credits
- **Prepayment** — funds loaded in advance
- **Total Usage** — the amount consumed this month

### Support

The Support section provides access to:

- **Tickets** — create and track support requests
- **Support Plans** — three tiers available:
  - **Basic** — community and documentation access
  - **Standard** — standard response SLA for business use
  - **Corporate** — dedicated support for enterprise customers
- Link to the platform status page

![img_15.png](img_15.png)

### Settings

Manage your organization configuration.

![img_16.png](img_16.png)

The Settings section has several tabs:

| Tab | Description |
|---|---|
| **Organization** | Organization name, ID, and member management |
| **Access Control** | Role-based access control for organization members |
| **Security** | Security settings |
| **Referrals** | Referral program |
| **Feature Preview** | Opt in to early-access features |

From the **Organization** tab you can see all members with their roles and invite new team members.

### API

Manage programmatic access to the platform.

![img_17.png](img_17.png)

| Tab | Description |
|---|---|
| **Tokens** | Personal Access Tokens — used for API authentication |
| **OAuth Applications** | Register OAuth applications |
| **Authorized Applications** | View and revoke access for third-party apps |
